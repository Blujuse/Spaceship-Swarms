## Dependencies
* SDL library
* SDL Image

## Target platforms
Support Windows x86/x64, GNU/Linux, OSX.

Instructions to build on Windows platforms:
1) Load .sln solution with Visual Studio;
2) Select Release configuration, and x64 Architecture;
3) Compile and run the project.
